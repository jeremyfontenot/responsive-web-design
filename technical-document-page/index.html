Diagnosing Problems

To accurately diagnose network problems, use the following command line tools:

  • ping - sends ICMP echo requests to a host.
  • tracert - traces the path taken by packets to reach a network host.
  • nslookup - queries the DNS to obtain domain name or IP address mapping.
  • netstat - displays network connections, routing tables, and a number of network interface statistics.
  • ipconfig/ifconfig - used to configure, manage and query TCP/IP network interface parameters.

Network Tools

Various software tools can assist in network troubleshooting:

  • Wireshark for packet analysis.
  • Nmap for network scanning.
